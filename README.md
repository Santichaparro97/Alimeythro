# Mundo Alimeythor

Sitio web estático replicando el diseño original (mobile-first, responsive).

## Estructura

```
mundo-alimeythor/
├── index.html          # Pestaña 1: Inicio (Home)
├── pages/              # Pestañas 2 a 6 (se irán agregando)
├── css/
│   └── styles.css
├── assets/             # Imágenes (logo, fotos, ilustraciones)
└── README.md
```

## Cómo verlo

Abrí `index.html` en el navegador, o serví la carpeta con cualquier servidor estático:

```bash
python -m http.server 8080
```
